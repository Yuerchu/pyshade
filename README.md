# PyShade

Build modern desktop apps in pure Python — Pydantic component DTOs compiled to shadcn/ui React,
running on the system WebView via [pytauri](https://github.com/pytauri/pytauri).

No bundled Chromium. No TCP port. No Node.js required at runtime.

> **Status: pre-alpha.** Nothing works yet — the architecture is being validated (milestone M0).
> See the design document (zh-CN): [docs/design.md](docs/design.md).
